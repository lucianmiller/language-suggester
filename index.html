JavaScript

Javascript is a high-level programming language that makes a website interactive. Learning this programing language is complex and time consuming, but very fun and interesting!

CSS

CSS (or Cascading Style Sheets) is a very commonly used programming language on the internet. It is used to style a webpage to make it look professional. Its a entry level program to learn, but it is very powerful and very fun!

HTML

HTML (or HyperText Markup Language) is the standard programming language to add text to webpages on the internet. It the very easy to learn, but powerful and necessary for every webpage to function.
